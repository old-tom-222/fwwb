启动后端服务：
```
mvn spring-boot:run
```
启动前端服务：
```
npm run serve
```
数据库内容：
```
数据库文件：数据库.json
```
推送GitHub：
```
git status(查看状态，可选)
git add .
git commit -m "引号里面写入这个修改的描述"
git push origin master
```
拉取GitHub
```
git status(查看状态，可选)
git fetch origin
git pull origin master
或git reset --hard origin/master(强制使本机代码与远程仓库一致)
```
